This is a paragraph in a div. A div is a container that helps divide code into sections.

We set a border for this div using Class = container. Border uses 3 arguments: linewidth shape and color.

For linewidth - use px to designate pixels. For shape you can use solid ; dotted ; or dashed. Color is self-explanatory.

Used margin in our CSS class to add space between end of page and beginning of div. You have spacing on the outside around all 4 sides of the div.

You can specify sides for margin. For example: margin-top ; margin-bottom ; margin-left ; margin-right.

Use padding to put space inside the box.

Same options for direction: -top ; -bottom ; -right ; -left.
